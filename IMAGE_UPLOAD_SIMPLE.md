## üìù STEP-BY-STEP UPLOAD

### Step 1: Create Folders
```
1. Go to your GitHub repository
2. Click "Add file" ‚Üí "Create new file"
3. Type "images/temp.txt" and commit
4. Repeat for "committee-images/temp.txt"
```

### Step 2: Upload Images
```
1. Go to each folder
2. Click "Add file" ‚Üí "Upload files"
3. Drag and drop your images
4. Click "Commit changes"
```

### Step 3: Wait & Refresh
```
Wait 2-3 minutes
Refresh your website
Images should appear!
```

## üí° QUICK TIPS

1. **All lowercase** for file names (who.jpg not WHO.jpg)
2. **No spaces** in file names
3. **Right size** = faster loading
4. **Good quality** but not too large (under 1MB)
